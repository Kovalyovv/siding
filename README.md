# Подбор металлического сайдинга для ворот и калитки

Каталог-агрегатор металлического сайдинга (2 источника: **mirkrovli31.ru** и
**belgorod.grandline.ru**) с удобными фильтрами по типу профиля, поверхности
и **цвету**, чтобы подобрать вариант для ворот/калитки под кирпич и забор дома.

**3321 вариант · 9 типов профиля · 64 цвета.** Данные — в [`data.json`](data.json).

## Возможности
- Фильтры: источник, тип профиля, поверхность (глянец / мат / текстура-дерево),
  палитра цветов (64 образца), цена ₽/м², поиск, сортировка.
- В каждой карточке: фото, образец цвета, цена, кнопка **«Источник»** и
  кнопка **«Характеристики»**.
- «Характеристики» подтягивают описание и таблицу характеристик прямо с
  сайта-источника (галерея всех фото товара — тоже).

## Запуск локально
```bash
python3 server.py          # → http://localhost:8000 (или 8001, если занято)
```
Локально характеристики тянутся через встроенный прокси `/api/specs`
(обходит CORS). На GitHub Pages, где бэкенда нет, тот же разбор выполняется
в браузере через публичный CORS-прокси.

## Онлайн
Сайт публикуется через GitHub Pages (статика: `index.html` + `data.json`).
Фото товаров подгружаются напрямую с сайтов-источников.

## Как собраны данные (воспроизведение)
```bash
python3 parse_siding.py      # парсинг mirkrovli31.ru   → siding.json
python3 parse_grandline.py   # парсинг grandline (48 стр.) → grandline.json
python3 fetch_galleries.py   # все фото по каждому товару Grand Line
python3 build_data.py        # свод + палитра hex        → data.json
```

## Файлы
| Файл | Назначение |
|---|---|
| `index.html` | сам сайт (фильтры, карточки, модалка характеристик) |
| `data.json` | сводный каталог (оба источника, цвета+hex, фото) |
| `server.py` | локальный сервер + прокси характеристик |
| `parse_siding.py` / `parse_grandline.py` | парсеры источников |
| `fetch_galleries.py` | догрузка всех фото по товарам Grand Line |
| `build_data.py` | объединение и палитра |
